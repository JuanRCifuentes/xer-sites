Les guides accompagnent l'utilisateur dans la réalisation d'une tâche précise qu'il souhaite accomplir, souvent à l'aide d'une suite d'étapes.
Rédiger un bon guide nécessite de réfléchir à ce que vos utilisateurs cherchent à faire.
