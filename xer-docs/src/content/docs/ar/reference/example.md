تُعدّ الصفحات المرجعية مثالية لتوضيح كيفية عمل الأشياء بعبارات موجزة وواضحة.
وبما أنها أقل اهتماماً بسرد قصة أو معالجة حالة استخدام محددة، فينبغي أن تقدّم عرضاً شاملاً لما توثّقه.
