मार्गदर्शिकाएँ उपयोगकर्ता को किसी विशिष्ट कार्य को पूरा करने में मदद करती हैं, अक्सर चरणों की एक शृंखला के माध्यम से।
एक अच्छी मार्गदर्शिका लिखने के लिए यह सोचना ज़रूरी है कि आपके उपयोगकर्ता क्या करना चाहते हैं।
