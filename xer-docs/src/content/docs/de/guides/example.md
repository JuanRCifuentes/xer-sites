Anleitungen führen Nutzer durch eine bestimmte Aufgabe, die sie erledigen möchten, oft anhand einer Reihe von Schritten.
Eine gute Anleitung zu schreiben bedeutet, darüber nachzudenken, was deine Nutzer erreichen wollen.
